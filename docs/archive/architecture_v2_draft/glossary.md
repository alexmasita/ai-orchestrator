Glossary
AI-Orchestrator

Adaptive reasoning and supervision system for autonomous development loops.

CDOS

Deterministic development operating system and mutation kernel.

Combo

Plugin-like composition of role models and runtime behavior.

Hot-swap

Unload one model runtime and load another while preserving correctness through externalized state.

Architect

Planning-oriented reasoning role.

Developer

Implementation-oriented reasoning role.

Runner

Verification execution role or service.

Cognitive Packet

Structured persisted artifact that allows reasoning continuity independent of model memory.

RepoMutationEngine

Stable mutation abstraction between orchestrator and repository execution authority.
