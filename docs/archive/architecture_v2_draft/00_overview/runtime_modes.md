Runtime Modes
Mode 1 — Advisory

AI plans only.

No autonomous patch application.

Mode 2 — Supervised Execution

AI can propose patches and request tools.
Human approval required before finalize.

Mode 3 — Guarded Autonomy

AI may iterate in draft and run verification.
Finalize remains gated.

Mode 4 — Bounded Autonomy

AI may iterate and finalize under policy restrictions.

This mode must only be enabled after governance, telemetry, and rollback maturity.
